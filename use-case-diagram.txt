╠════════════════════════════════════════════════════════════════════════════════════════════════════════════════════╣

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ CUSTOMER USE CASES (LEFT SIDE)                                                                                      │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ 1. Browse Produk              → Melihat produk tanpa login                                                          │
│ 2. Lihat Gallery Produk       → Browse produk berdasarkan kategori                                                  │
│ 3. Detail Produk              → Melihat detail produk (harga, deskripsi, stok)                                      │
│ 4. Search Produk              → Mencari produk berdasarkan nama/barcode                                             │
│ 5. Register                   → Registrasi akun customer                                                            │
│ 6. Login                      → Login ke akun                                                                       │
│ 7. Update Profil              → Update data profil (nama, email, phone)                                             │
│ 8. Kelola Alamat              → Manage alamat pengiriman (address, city, province)                                  │
│ 9. Tambah ke Cart             → Menambahkan produk ke keranjang                                                     │
│10. Kelola Cart                → Edit qty, remove item dari cart                                                     │
│11. Checkout                   → Proses checkout order                                                                │
│12. Pilih Alamat Pengiriman    → Pilih alamat dari profil atau input baru                                            │
│13. Pilih Metode Pengiriman    → Pilih pengiriman (Reguler/Express/Same Day)                                         │
│14. Pilih Pembayaran           → Pilih metode pembayaran (COD/Transfer/QRIS/E-Wallet)                                │
│15. Upload Bukti Pembayaran    → Upload bukti untuk Transfer/QRIS/E-Wallet                                           │
│16. WhatsApp Admin             → Redirect ke WhatsApp admin untuk COD                                                │
│17. Track Order                → Melacak status order                                                                │
│18. Lihat Riwayat Transaksi    → Melihat history transaksi customer                                                  │
│19. Logout                     → Keluar dari akun                                                                    │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ ADMIN/CASHIER USE CASES (RIGHT SIDE)                                                                                │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ 1. Login                      → Login ke sistem                                                                     │
│ 2. View Dashboard             → Melihat overview transaksi, revenue, profit                                         │
│ 3. Manage Products            → CRUD produk (create, read, update, delete)                                          │
│ 4. Manage Category            → CRUD kategori produk                                                                │
│ 5. Manage Customer            → CRUD data customer                                                                  │
│ 6. Manage User & Role         → Manage user, role, dan permission                                                   │
│ 7. POS Checkout               → Halaman Point of Sales untuk transaksi                                              │
│ 8. Search Product (POS)       → Cari produk untuk transaksi                                                         │
│ 9. Add to Cart (POS)          → Tambah produk ke cart POS                                                           │
│10. Process Checkout           → Proses transaksi checkout                                                           │
│11. Print Invoice              → Cetak invoice/struk setelah checkout                                                │
│12. Transaction History        → Lihat semua riwayat transaksi                                                       │
│13. Filter & Search Transaksi  → Filter transaksi by date/customer/cashier                                           │
│14. View Detail Transaksi      → Lihat detail transaksi                                                              │
│15. Reprint Invoice            → Cetak ulang invoice                                                                 │
│16. Profit Report              → Lihat laporan keuntungan                                                            │
│17. Analytics Dashboard        → Lihat analytics penjualan                                                           │
│18. Update Profil              → Update profil admin/cashier                                                         │
│19. Change Password            → Ganti password                                                                       │
│20. Logout                     → Keluar dari sistem                                                                  │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ SHARED USE CASES (CENTER - CONNECTING BOTH ACTORS)                                                                  │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ • Search Product              → Customer browse & Admin POS search                                                  │
│ • Product Detail              → Customer view & Admin product management                                            │
│ • Add to Cart                 → Customer shopping cart & Admin POS cart                                             │
│ • Manage Cart                 → Customer cart management & Admin POS cart management                                │
│ • Checkout                    → Customer checkout flow & Admin POS checkout                                         │
│ • Select Address              → Customer delivery address & Admin customer selection                                │
│ • Select Shipping             → Customer shipping method & Admin shipping calculation                               │
│ • Select Payment              → Customer payment selection & Admin payment verification                             │
│ • Upload Proof                → Customer upload payment proof & Admin verify proof                                  │
│ • WhatsApp Admin              → Customer contact admin & Admin receive messages                                     │
│ • Track Order                 → Customer track order & Admin update order status                                    │
│ • Transaction History         → Customer view own history & Admin view all transactions                             │
│ • Logout                      → Both actors logout from system                                                      │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 🛒 CUSTOMER (Left Side)                                                                                             │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ • External user who browses and purchases products                                                                  │
│ • Can browse products without login                                                                                 │
│ • Must register/login to checkout                                                                                   │
│ • Payment methods: COD, Transfer, QRIS, E-Wallet                                                                    │
│ • Can track order status and view purchase history                                                                  │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 👤 ADMIN/CASHIER (Right Side)                                                                                       │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ • Internal user who manages the system                                                                              │
│ • Full access to all management features                                                                            │
│ • Handles POS transactions and checkout process                                                                     │
│ • Manages products, categories, customers, and users                                                                │
│ • Views reports, analytics, and profit tracking                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 🖥️ APLIKASI KASIR / POS SYSTEM (Center)                                                                             │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ • Central system connecting Customer and Admin/Cashier                                                              │
│ • Handles all business logic and data processing                                                                    │
│ • Manages authentication, authorization, and sessions                                                               │
│ • Processes transactions and generates invoices                                                                     │
│ • Tracks orders and maintains transaction history                                                                   │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
